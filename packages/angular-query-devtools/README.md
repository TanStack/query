# TanStack Angular Query Devtools

Developer tools for [@tanstack/angular-query-experimental](https://www.npmjs.com/package/@tanstack/angular-query-experimental).

See the [Angular Devtools documentation](https://tanstack.com/query/latest/docs/framework/angular/devtools).

## Installation

```bash
npm install @tanstack/angular-query-devtools
```

Peer dependencies: `@angular/core`, `@angular/common`, and `@tanstack/angular-query-experimental`.
