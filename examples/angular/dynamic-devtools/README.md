# TanStack Query Angular dynamic devtools example

Devtools load automatically in development. In production builds, press **⌘ Ctrl Shift D** (macOS) to load them on demand.

To run:

- From the repo root: `pnpm install` then `pnpm --filter @tanstack/query-example-angular-dynamic-devtools start`
- From this folder: `pnpm start`

For a production-like devserver (so you can try the shortcut), use `ng serve --configuration production`.
